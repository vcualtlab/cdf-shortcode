# ALT Lab .cdf Shortcode

Requires at least: 3.0.1
Tested up to: 4.2.2

Shortcode to embed Wolfram Alpha .cdf files

## Description

Provides shortcode to embed Wolfram Alpha .cdf files. Depends on Wolfram Alpha's browser plugin which does not work in Chrome.

`[wcdf cdf="" image="" width="" height=""]`

| Attribute         | Default    | Description   
| :---------------- | :--------- | :------------- 
| cdf               | ''         | url to your .cdf file
| image             | ''         | url to a backup image for those without plugin or unsupported browsers
| width             | '824'      | width in pixels
| height            | '652'      | height in pixels
